## Getting Started

### First Login

1. Open your web browser and navigate to the application URL (e.g., `http://your-domain.com:3000`)
2. You'll see the login page
3. Enter your credentials:
   - **Default admin**: Username: `admin`, Password: `admin123`
   - ⚠️ **Important**: Change the default password after first login!
4. Click **Login**

## Dashboard Overview

The main dashboard is your central hub for navigating the application.

### Dashboard Cards

- **Agents**: Manage your modular agents
- **Pipelines**: Create and manage data processing pipelines
- **Monitoring**: View real-time system metrics
- **Audit Logs**: Track all system activities
- **Admin**: User and role management (admin only)

---

## Managing Agents

Agents are modular components that perform specific data processing tasks.

### Agent Types

- **Validator**: Validates data against rules
- **Analyzer**: Analyzes and extracts insights
- **Enricher**: Adds additional data
- **Transformer**: Changes data structure
